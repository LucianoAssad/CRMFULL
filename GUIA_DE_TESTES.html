🟢 P0 — Base e Correções

✅ Dashboard com métricas

Caminho: Menu lateral → Dashboard
  • Verificar cards: Total de leads, Conversas abertas, Oportunidades, Vendas, Valor ganho (verde), Valor perdido (vermelho)
  • Gráfico "Novos contatos por dia" — últimos 30 dias
  • Tabela "Performance de Campanhas"
  • Trocar abas UTM: Campanha / Source / Medium / Conjunto / Anúncio — tabela muda
  • Filtrar por empresa e por período (De / Até)

✅ Pipeline — Editar coluna inline

Caminho: Menu lateral → Pipeline
  • Clicar no nome de uma coluna (ex: "Novo") — campo de edição aparece
  • Digitar novo nome e pressionar Enter — salva
  • Pressionar Escape — cancela sem salvar

🔵 P1 — Features de Atendimento

✅ Mensagens rápidas via "/"

Caminho: Atendimento → selecionar conversa → campo de mensagem
  • Digitar / — lista de respostas rápidas aparece acima do input
  • Continuar digitando para filtrar (ex: /ola)
  • Usar ↑ ↓ para navegar, Enter para selecionar
  • Clicar numa opção com o mouse

✅ Variáveis do contato via "\"

Caminho: Atendimento → selecionar conversa → campo de mensagem
  • Digitar \ — popup de variáveis aparece
  • Opções: Nome, Primeiro nome, Telefone, Email, CPF, Cidade, Estado
  • Clicar numa variável — substitui \ pelo valor real do contato
  • Pressionar Escape — fecha sem substituir

✅ Gravar áudio (PTT)

Caminho: Atendimento → selecionar conversa → ícone 🎙️ (campo vazio)
⚠️ Pré-requisito: o campo de mensagem deve estar vazio para aparecer o ícone de microfone
  • Clicar no ícone de microfone — browser solicita permissão de microfone
  • Permitir — cronômetro aparece (00:00)
  • Falar a mensagem
  • Clicar em Send (ícone verde) — envia
  • Clicar no quadrado (■) — cancela sem enviar

✅ Catálogo de produtos no chat

Caminho: Atendimento → selecionar conversa → ícone 📦
  • Clicar no ícone de caixinha na barra de ferramentas do chat
  • Dialog com lista de produtos abre
  • Buscar produto pelo nome
  • Clicar num produto — texto formatado é inserido no campo de mensagem
  • Editar o texto e enviar normalmente
💡 Também disponível em: ícone ⋮ (três pontos) → "Catálogo de produtos"

✅ Agendar mensagem

Caminho: Atendimento → conversa → ícone ⋮ → "Agendar mensagem"
  • Clicar nos três pontos no header da conversa
  • Clicar em "Agendar mensagem"
  • Preencher: texto da mensagem, data e hora
  • Clicar em "Agendar" — mensagem salva para envio futuro

✅ Saudação automática

Caminho: Menu lateral → Configurações → seção "Saudação automática"
  • Ativar o toggle Ativo/Inativo
  • Adicionar até 5 mensagens com delay configurável (segundos)
  • Usar variáveis: {{nome}}, {{telefone}}
  • Clicar em Salvar

✅ CEP autocomplete

Caminho: Atendimento → conversa → painel direito → "Cadastro completo" → campo CEP
  • Digitar um CEP válido (ex: 22041001)
  • Rua, Bairro, Cidade e Estado preenchem automaticamente via API ViaCEP

✅ Gênero do lead

Caminho: Atendimento → conversa → painel direito → "Cadastro completo"
  • Campo "Gênero": Não definido / Masculino / Feminino / Outro
  • Selecionar e clicar "Salvar alterações"

✅ Token CAPI Meta e Vínculo por canal

Caminho: Menu lateral → Conversões → aba Configurações → Meta Ads
  • Campo "Token de acesso CAPI" com toggle mostrar/ocultar
  • Select "Vincular ao número/instância WhatsApp" — vincula canal ao pixel

🟣 P2 — Novas Funcionalidades

✅ Agendamentos — Calendário

Caminho: Menu lateral → Agendamentos
  • Calendário mensal: navegar meses com ← →, clicar "Hoje" para voltar
  • Clicar num dia → painel direito exibe agendamentos daquele dia
  • Duplo clique num dia → abre modal de novo agendamento naquela data
  • Botão "Novo agendamento" → formulário completo: título, tipo, lead, responsável, local, link de reunião, lembrete
  • Tipos: 🤝 Reunião · 📞 Ligação · 🏠 Visita · ✅ Tarefa · 🔔 Follow-up
  • Marcar como Concluído ou Cancelar diretamente no card
  • Aba Lista: exibe todos os agendamentos cronologicamente
💡 Também disponível no chat: ícone ⋮ → "Criar agendamento" (vincula à conversa atual)

✅ Agendamentos no painel do lead

Caminho: Atendimento → conversa → painel direito → seção "Agendamentos"
  • Lista de agendamentos do lead com status
  • Marcar como concluído / cancelar inline
  • Criar novo agendamento direto no painel

✅ Grupos WhatsApp

Caminho: Menu lateral → Grupos WhatsApp
  • Exibe aviso se não houver canal Evolution API ou WPPConnect conectado
  • Botão "Registrar grupo" → nome, canal, JID do grupo, descrição
  • Editar e remover grupos cadastrados

🟠 P3 — Integrações e Crescimento

✅ Afiliados

Caminho: Menu lateral → Afiliados
  • KPIs no topo: afiliados, ativos, indicações, convertidas, comissão total, pendente pagar
  • Aba Afiliados: cadastrar com código (auto-gerado), % comissão, copiar link de indicação
  • Aba Indicações: registrar indicação, mudar status (pendente → contatado → convertido → perdido)
  • Quando convertido: inserir valor da venda, clicar "Marcar pago" para quitar comissão

✅ Base de Conhecimento

Caminho: Menu lateral → Base de Conhecimento
  • Botão "Novo artigo" → título, categoria, tags, conteúdo em Markdown
  • Toggle "Visível para clientes" — artigo público ou privado
  • Buscar artigo por nome, conteúdo ou tag
  • Filtrar por categoria: geral, produto, atendimento, técnico, financeiro, vendas, políticas, FAQ
  • Clicar no card → abre modal de leitura com contador de visualizações
  • Editar e excluir artigos

✅ Comunidade

Caminho: Menu lateral → Comunidade
  • Página com roadmap de features futuras
  • Links de suporte: WhatsApp, YouTube, grupo de usuários
  • Histórico de últimas atualizações da plataforma

✅ Áudio WABA MP3 (OGG/Opus)

Caminho: Atendimento → conversa em canal Cloud API → ícone 🎙️
  • Para canais WhatsApp Cloud API oficial, o áudio é automaticamente convertido para OGG/Opus
  • Formato compatível com a API oficial do WhatsApp (WABA)
  • Para outros canais, envia em WebM como fallback

🐛 Como Reportar Bugs

  1. Print da tela com o erro visível
  2. Caminho percorrido para reproduzir o problema
  3. Console do browser: pressionar F12 → aba Console → copiar mensagem vermelha
  4. Descrever o comportamento esperado vs. o que aconteceu
